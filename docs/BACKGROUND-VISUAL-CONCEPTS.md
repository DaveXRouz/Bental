## 1. Quantum Field Background

### Visual Description

Imagine looking at a deep space view through a futuristic holographic display. Three massive, translucent orbs drift slowly across a dark navy gradient backdrop, each glowing with ethereal light.

**What You See:**
- **Base Layer:** Deep space gradient transitioning from nearly black (#0a0e1a) at the edges to dark navy (#1a1f35) in the center
- **Orb 1 (Top Right):** Large blue-purple sphere (600x600) floating near the top right, slowly moving right and up over 20 seconds
- **Orb 2 (Bottom Left):** Green-cyan sphere (500x500) near bottom left, drifting left and down over 30 seconds
- **Orb 3 (Middle Right):** Purple-blue sphere (400x400) in the middle right area, floating independently over 16 seconds
- **Grid Overlay:** Ultra-subtle grid pattern (3% opacity) suggesting advanced technology
- **Glow Effect:** All orbs pulse softly between 15-35% opacity over 3-second cycles

**Movement Character:**
- Slow, weightless floating
- Organic, non-linear paths
- Gentle scaling (1.0 to 1.15x)
- Smooth transitions
- Zero sharp edges or angles

**Mood:**
- Premium and sophisticated
- Calm and mysterious
- Futuristic yet approachable
- Professional and trustworthy

**Inspired By:**
- Holographic displays
- Quantum computing visualizations
- Premium fintech apps
- Space telescope imagery
- Bioluminescent organisms

## 2. Data Stream Background

### Visual Description

Picture a high-tech trading floor where streams of data cascade continuously down transparent displays. Five vertical lines of different colors flow endlessly from top to bottom, each representing different data channels.

**What You See:**
- **Base Layer:** Gradient from deep black (#050810) to dark blue-gray (#0f1729)
- **Stream 1 (Left 10%):** Bright blue stream, 150px tall, flows down in 8 seconds
- **Stream 2 (Right 15%):** Emerald green stream, 200px tall, flows down in 10 seconds

- **Ambient Orbs:** Two large glowing orbs pulse in the background (similar to Quantum)

**Movement Character:**
- Continuous linear descent
- Varying speeds (creates rhythm)

- Never stops or pauses

**Mood:**

- Professional and technical

**Inspired By:**

## Customization Possibilities

### Easy Customizations

**Color Changes:**
```typescript
// All backgrounds use rgba() colors
// Simply change the color values in the component
colors: ['rgba(YOUR_R, YOUR_G, YOUR_B, 0.3)', ...]
```

**Speed Adjustments:**
```typescript
// Change duration values
duration: 15000, // 15 seconds instead of 20
```

**Opacity Control:**
```typescript
// Wrap background with opacity view
<View style={{ opacity: 0.7 }}>
  <Background />
</View>
```

### Advanced Customizations

**Add Overlay:**
```typescript
<Background />
<LinearGradient
  colors={['transparent', 'rgba(0,0,0,0.5)']}
  style={StyleSheet.absoluteFill}
/>
```

**Blend Backgrounds:**
```typescript
<QuantumFieldBackground />
<View style={{ opacity: 0.3 }}>
  <DataStreamBackground />
</View>
```

**Theme Integration:**
```typescript
const colors = useThemeColors();
// Pass custom colors to background
<Background primaryColor={colors.primary} />
```
